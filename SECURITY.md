# Security

This repository does not publish operational code or implementation materials.
Even so, maintainers prefer that potential disclosure concerns be reported
privately rather than raised in public issues.

If you believe a document in this repository unintentionally creates security,
confidentiality, or disclosure risk, contact the maintainers directly and
include:

- the file name
- the passage of concern
- a short explanation of the risk

Please do not post potentially sensitive disclosure concerns publicly before
maintainers have had an opportunity to review them.
