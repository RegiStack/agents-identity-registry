# Thesis

Autonomous agents are not static artifacts. They can change through new tools,
new instructions, new environments, new dependencies, and new operator
contexts. As autonomy increases, the distinction between observing behavior and
establishing identity becomes more important.

Behavior alone can be suggestive without being sufficient. Observation may show
what appears to be happening, but it does not necessarily establish whether the
acting system remains the same governed system that was previously reviewed,
authorized, or trusted.

Registack is conceptually grounded in the view that identity may become the
primary control plane for autonomous agents. In that view, execution permission
should not rest only on policy, scanning, or monitoring. It should depend on
the continued consistency of an agent with an authorized context for action.

This repository does not disclose how such a model is implemented.
