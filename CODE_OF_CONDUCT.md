# Code Of Conduct

This repository is maintained as a professional public record.

Participants are expected to:

- communicate respectfully
- stay precise and constructive
- avoid speculation presented as fact
- respect the repository's intentionally limited scope

The following are not acceptable:

- harassment or abusive conduct
- hostile or inflammatory language
- disclosure pressure for non-public technical details
- repeated attempts to push the repository beyond its stated publication scope

Maintainers may moderate discussions, close threads, or restrict participation
when needed to preserve a respectful and focused environment.
